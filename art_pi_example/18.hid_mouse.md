# 18.hid_mouse

此示例应用演示了 Zephyr 项目如何使用 USB 人机接口设备 (HID) 驱动程序。这个非常简单的驱动程序将一块带按钮的开发板枚举为一个鼠标，该鼠标具有鼠标左键，并且可选地（取决于开发板上的按钮数量）具有鼠标右键，可进行 X 轴和 Y 轴移动。如果 USB 外设驱动程序支持远程唤醒功能，则当总线处于挂起状态时，每次单击按钮都会执行唤醒请求。此示例位于 Zephyr 项目树中的[samples/subsys/usb/hid-mouse目录下。](https://github.com/zephyrproject-rtos/zephyr/blob/main/samples/subsys/usb/hid-mouse)

```
west build -p always -b art_pi -d build 18.hid_mouse -- \
-DCONFIG_BOOTLOADER_MCUBOOT=y \
-DCONFIG_MCUBOOT_SIGNATURE_KEY_FILE="\"$HOME/zephyrproject/bootloader/mcuboot/root-rsa-2048.pem\""
```
