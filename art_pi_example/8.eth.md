# 8.eth

功能：dhcp,打印连接信息

## 编译

```
  west build -p always -b art_pi -d build 8.eth -- \
  -DCONFIG_BOOTLOADER_MCUBOOT=y \
  -DCONFIG_MCUBOOT_SIGNATURE_KEY_FILE="\"$HOME/zephyrproject/bootloader/mcuboot/root-rsa-2048.pem\""
```
