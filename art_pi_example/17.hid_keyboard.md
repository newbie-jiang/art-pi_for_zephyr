# 17.hid_keyboard

此示例应用程序演示了使用新的实验性 USB 设备堆栈的 HID 键盘实现。

**把板子变成 USB 键盘”的最小工程**。overlay 决定端点缓冲/轮询参数；真正的键码格式在代码的 HID 报告描述符里，输入事件来自板上外设或你自己造的事件。



```
west build -p always -b art_pi -d build 17.hid_keyboard -- \
-DCONFIG_BOOTLOADER_MCUBOOT=y \
-DCONFIG_MCUBOOT_SIGNATURE_KEY_FILE="\"$HOME/zephyrproject/bootloader/mcuboot/root-rsa-2048.pem\""
```
