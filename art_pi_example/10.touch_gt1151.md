# 10.touch_gt1151

功能：gt1151五点触摸

zephyr驱动不支持gt1151,需要参考gt911来实现，参考   zephyr增加自己的驱动(官方路径).md



```
  west build -p always -b art_pi -d build 10.touch_gt1151 -- \
  -DCONFIG_BOOTLOADER_MCUBOOT=y \
  -DCONFIG_MCUBOOT_SIGNATURE_KEY_FILE="\"$HOME/zephyrproject/bootloader/mcuboot/root-rsa-2048.pem\""
```
