## 11.lvgl

功能：此示例应用程序在屏幕中央显示“Hello World”，并在底部显示一个每秒递增的计数器。支持gt1151触摸中央的Hello World来清除计时



```
  west build -p always -b art_pi -d build 11.lvgl -- \
  -DCONFIG_BOOTLOADER_MCUBOOT=y \
  -DCONFIG_MCUBOOT_SIGNATURE_KEY_FILE="\"$HOME/zephyrproject/bootloader/mcuboot/root-rsa-2048.pem\""
```
