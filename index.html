    ART-PI for Zephyr¶

    Support hardware¶

    Basic development board¶

    • ✓ LED
    • ✓ UART
    • ✓ BUTTON
    • ✓ FMC SDRAM
    • ✓ LTDC 800x480
    • ✓ TOUCH GT1151
    • ✓ QSPI W25Q64JV
    • ✓ SPI W25Q128JV
    • ✓ SDMMC SD
    • ✓ USB

    Industrial expansion board¶

    • ✓ BUZZER
    • ✓ RS232
    • ✓ RS485
    • ✓ ETH
    • ○ FDCAN
    • ○ DC_IN

    Common picocom shortcut keys

    • Ctrl + A, Ctrl + Q → 挂起串口，不退出
    • Ctrl + A, Ctrl + Z → 帮助菜单
    • Ctrl + A, Ctrl + X → 退出并关闭设备
